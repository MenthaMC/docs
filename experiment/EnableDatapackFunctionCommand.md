# 配置 - enable_datapack_function_command

> Experiment类的配置可能会有点bug，请谨慎使用。

本配置位于mint_global.toml，来自于上游Mint

```toml
[experiment.enable_datapack_function_command]
    enabled = false
```

该配置用于修复数据包指令以及Function指令，让其Function能够正常使用。

## 字段

- enabled - 用于开关此配置
